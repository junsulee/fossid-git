# fossid-git
FossID GIT Integration Script (Beta)

## Prerequisites
- install jq package
> sudo apt-get install jq

> yum install jq

## Usage
> ex) ./fossid-git.sh https://yourfossidurl/webapp/api.php username userapikey projectname scanname - giturl gitbranchname
